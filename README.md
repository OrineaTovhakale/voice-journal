# Voice Journal — React Native Audio Recorder

**Full-featured voice recording app** • Custom titles • Search by title/date • Beautiful dark UI • Local storage • 100% working

**Submitted by:** Orinea Tovhakale  
**Date:** December 2025  
**Status:** Complete 

### Features Implemented
- Record audio with live timer
- Save recordings with **custom titles**
- **Search** by title or date
- Play/Pause recordings
- Delete with confirmation
- Gorgeous dark modern design
- Recordings **survive app close** (AsyncStorage)
- Zero crashes — fully stable
